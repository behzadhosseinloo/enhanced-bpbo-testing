# Enhanced BPBO for Test Data Generation

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)(https://colab.research.google.com/drive/1s9isoBwj2gU0bYgw6BAnJYFUWmP_MGT5?usp=drive_link)}

This repository contains the source code and experimental data for the paper:
**"Test Data Generation for Improved Path Coverage and Diversity Using Birds of Prey-Based Optimization"**

## 📋 Contents

- `src/` - Python implementation of Enhanced BPBO algorithm
- `benchmarks/` - TriangleType benchmark program
- `notebooks/` - Google Colab notebook for running experiments
- `results/` - Output results and convergence data
- `figures/` - Generated figures (convergence curves)

## 🚀 Quick Start

1. Open the notebook in Google Colab:
  https://colab.research.google.com/drive/1s9isoBwj2gU0bYgw6BAnJYFUWmP_MGT5?usp=drive_link

2. Run all cells

3. Results will be displayed and saved automatically

## 📊 Results

- **Path Coverage:** 4/4 paths (100%)
- **Convergence:** 7 generations
- **Best Test Case:** [1, 1, 1] (Equilateral triangle)
